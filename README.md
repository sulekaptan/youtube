# YouTube

## Run Locally 

Clone the project  

~~~bash  
git clone https://github.com/sulekaptan/youtube.git
~~~

Go to the project directory  

~~~bash  
cd youtube
~~~

Install dependencies  

~~~bash  
npm install
~~~

Start the project  

~~~bash  
npm start
~~~
